# taxer

## Cài đặt package
```
npm install
```

### Cài đặt json-server
```
npm install -g json-server
```

### Chạy json server
```
json-server -w .\data\db.json
```

### Chạy project
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
